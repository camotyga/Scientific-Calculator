# Scientific Calculator

A scientific calculator built in Java that performs both basic arithmetic and advanced mathematical operations through a simple graphical user interface.

## Features

* Basic arithmetic operations
   * Addition
  * Subtraction
  * Multiplication
  * Division
    
* Scientific functions
  * Square root
  * Powers and exponents
  * Trigonometric functions (sin, cos, tan)
  * Logarithms
  * Factorial
  * Percentage
    
* Clear and delete functions
* User-friendly interface
* Input validation to prevent invalid calculations

## Technologies Used

* Java
* Java Swing (GUI)
* Object-Oriented Programming (OOP)

## How to Run

1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or NetBeans).
3. Compile and run the main Java file.
4. Start performing calculations.
